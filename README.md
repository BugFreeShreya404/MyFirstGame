# My First GitHub Project

This is my first project uploaded on GitHub as part of learning Git and GitHub.

## Description
This repository contains a simple file created for practice and understanding how to upload projects to GitHub using Git Bash.

## Technologies Used
- Python (basic)
- Git
- GitHub

## How to Run
1. Download or clone this repository
2. Open the file in any code editor (VS Code recommended)
3. Run the file using:
   python filename.py

(Replace `filename.py` with the actual file name)

## Purpose
- Learn Git and GitHub basics
- Practice version control
- Build a beginner-friendly GitHub profile
